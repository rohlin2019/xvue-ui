# xvue-ui
xvue-ui是一款基于vue.js开发的响应式框架,包含丰富的css和js组件。

## Official Website
[xvue-ui official website](http://www.xvue.cn).

## Installation

```shell
npm install xvue-ui --save
```

## Usage

import all components.

``` javascript
import Vue from 'vue';
import xvueui from 'xvue-ui';

Vue.use(xvueui);
```
